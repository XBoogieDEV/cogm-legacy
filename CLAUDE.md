# Claude Code Guidelines

## Commit Messages
- Do NOT include "Generated with Claude Code" footer
- Do NOT include "Co-Authored-By: Claude" attribution
- Use standard conventional commit format
- Keep messages concise and descriptive

## Code Comments
- Do NOT add comments mentioning Claude or AI assistance
- Use standard code documentation practices
- Focus on explaining "why" not "what"

## Pull Requests
- Do not include AI-generated badges or footers
- Follow standard PR templates and conventions
